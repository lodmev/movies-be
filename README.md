## Project setup
### Clone this repository
```bash
$ git clone https://github.com/lodmev/movies-be
```
### Go to folder
```bash
$ cd movies-be
```
### Up the containers 
```bash
$ docker compose up -d
```
### Run folow commands 
```bash
$ pnpm install
```
```bash
$ pnpm prisma migrate deploy
```
```bash
$ pnpm prisma generate
```
```bash
$ pnpm start
```
